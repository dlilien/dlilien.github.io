read_dzg¶

usage: read_dzg.py [-h] [-t_srs {sps,wgs84,ll,EPSG:3031}] [-o O] fn [fn ...]

Positional Arguments¶

fn files with extension dzg to process

Named Arguments¶

-t_srs

Possible choices: sps, wgs84, ll, EPSG:3031

Convert to this reference

Default: “sps”

-o Name for combined output

pygssi
