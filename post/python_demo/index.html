Python Brownbag

Oct 16, 2017 1 min read

I’m giving a short seminar on python for matlab users this Friday. I plan to give a short presentation on why grad students in Earth and Space Sciences might want to switch from Matlab, followed by prepared demos where people can follow along.

I’ll also introduce git, since I think it is probably more important for productivity than the matlab/python choice. Hopefully I’ll have a few open-source converts by the end of the week!

Everything I’m presenting is on github. There are two sets of slides, one on Python vs. Matlab and one giving a whirlwind intro to git and the package managers.
